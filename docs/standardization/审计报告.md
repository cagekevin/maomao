## 一、工程化配置缺失（高优先级）

| 问题 | 现状 | 建议 |
|---|---|---|
| 无任何 JS/TS lint 配置 | 全仓无 `.eslintrc*`、无 `eslint` 依赖 | 根 + `localTool` 各加 ESLint（推荐 flat config + `typescript-eslint`）。首轮只开"无错误"规则，避免对混淆 `App.js` 误报（在 `ignorePatterns` 排除 `src/App.js`、`src/vendor/`） |
| 无格式化工具 | 无 `.prettierrc*`、无 Prettier | 加 Prettier，配合 `.editorconfig` 统一缩进/换行/尾随空格 |
| 无 `.editorconfig` | 混 JS/TS/Python/`.cjs`/`.ps1`，易产生 CRLF/LF 混用与缩进漂移 | 增加 `.editorconfig`（`*.py`=4空格、`*.{js,ts,cjs}`=2空格、换行 LF、`trim_trailing_whitespace=true`） |
| Python 无任何工具链配置 | 无 `pyproject.toml`、`ruff`/`flake8`/`black` | 两个 Python 服务各加 `pyproject.toml`（放 ruff 配置）；`ai-image-agent` 尤其缺失 |

## 三、依赖与可复现性（中高优先级）

| 问题 | 现状 | 建议 |
|---|---|---|
| Python 依赖版本未固定 | `apimart-gateway/requirements.txt` 全用下限（`fastapi>=0.110` 等），无上界/锁文件，安装结果不可复现 | 至少加 `==` 或兼容上界；条件允许用 `uv`/`pip-tools` 生成 `requirements.lock.txt` |
