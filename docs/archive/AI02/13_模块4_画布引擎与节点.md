## 一、运转图景

模块4 是画布表现层：ReactFlow 渲染节点/连线，处理拖放(`Lr`=onDrop)、连线(`xn`=onConnect)、右键菜单(`Cn`/`wn`)、点击(`kn`)、连线落空(`Fr`=onConnectEnd)，并承载 16+ 种节点组件（ImageNode/PromptNode/VideoNode/3D 导演台等）。

**性能降级**：节点>20 隐藏非可见、>300 降透明度（`.is-large-canvas` / `pan-performance-mode`，L31232 CSS）。


### 3.3 画布状态持久化
```
ReactFlow onChange → （非 CustomEvent）
  → Cr(projectId) → Q(localforage) 存 canvas-state-v1-{id}   （见模块1 / 模块5）
```

```
onPaneContextMenu(Cn@L31933) / onNodeContextMenu(wn@L31947)
```
