## X6. 跨层一致性风险（汇总）

1. **双数据源**：`transitItems`(内存) vs `resources`(持久) — 素材 Tab 显示 resources（X2）。
2. **资源 ID 冲突**：`resourceAdded` 用 `Date.now()`；rescan 用 `local-{folder}-{name}` — 同一文件可能两条记录（ARCHITECTURE X3.2）。
3. **base URL 不一致**：`Hr`(L1732)=localEngineBase 按开关；`vv`(L42808)=硬编码 18080。`USE_LOCAL_ENGINE=false` 时 `Hr`=9004 但 `vv` 仍 18080 → 文件操作走错端口（TASKS P2-5 确认）。
4. **rescan URL 格式**：后端 `toAbsoluteFileUrl`(resources.ts L31) 已补全绝对路径 ✅；上传侧 `files.ts` L42 返回相对路径，依赖前端兜底（P0-1）。
5. **缩略图**：`/files/{sub}/.thumbnails/thumb_*`(files.ts L131) ✅；rescan 跳过 `.thumbnails` ✅。
