## 一、安装与基础操作

### 1. 下载与安装

- 官网（一锚点 cc / yimao.cc）点击「免费下载」，选最新版本下载「多合一」压缩包（支持 Mac / Windows）。
- 解压后目录结构（以 1.2.0 为例）：

```
猫猫AI画布1.2.0
├── dist/                              <-- 浏览器插件的核心文件
├── data/                             <-- 你的素材 / 生成的内容
猫猫AI画布1.2.0本地引擎.exe            <-- Windows 版本地引擎，双击运行
猫猫AI画布引擎Mac端Arm版              <-- macOS 本地引擎，双击运行
猫猫AI画布引擎Mac端Intel版           <-- macOS 本地引擎，双击运行
```

- **macOS 支持的最低版本是 macOS 12**。
- 双击本地引擎运行，会默认打开一个网页（黑窗口=本地引擎，**切勿关闭**，软件/插件都依赖它）。只想当 AI 画布用，打开即可直接使用。
- 若软件界面不小心关了，可重新输入地址访问，或点右下角工具图标快速进入。

### 2. 当浏览器插件使用（可选）

- 进入浏览器「扩展程序」页面，**先打开右上角「开发者模式」开关**。
- 点击「加载已解压的扩展程序」，选择安装目录里的 **`dist` 文件夹**（必须是 dist 本身）。
- 加载成功后可点亮图钉固定插件。插件与软件**同源**，两边内容互通。

### 3. 登录注册

- 右上角一键注册/登录，输入手机号和验证码即可；注册送一些算力。
- **v1.3.5 起**在原有方式基础上，额外支持**账号密码登录**。

### 4. 基础画布操作

- **添加节点**：在画布空白处**右键点击**，呼出添加菜单，选择所需节点（文本、生图、视频等）。
- **右键 → 小工具**：v1.3.3 起新增画布小工具入口（如对比工具、人脸打码等，见第四节）。

### 5. 一个网站多账号登录（薅羊毛必备）

适用于同站多开、提升免费额度的场景：

1. **未登录状态**下，点左上角「保存环境」，改名为「未登录」。
2. 登录**账号 A** → 点「保存环境」存为「账号 A 的环境」。
3. **关键**：切回「**未登录**」环境。
4. 登录**账号 B** → 保存为「账号 B 的环境」。
5. 检查各环境是否对应不同账号——完成多开。

> 原理：保存/切换「浏览器环境」实现多账号隔离，绝大部分网站适用。

## 二、实战：从文本到视频

1. 右键放置**文本节点**，选择模型（提供多种免费文本节点）。
2. 让 AI 写生图提示词，微调后把输出连入**生图节点**，选模型（约 5 分钱/张）点击生成；等待时界面轮播技巧 tips；完成后图片进入「资产」。
3. 拖入**视频节点**连线。生视频消耗**特惠币**（右上角第二个数字），注册不送需充值。
4. 选视频模型——工具提供清晰度/速度/稳定性三维对比；第二梯队最低约 3 分/秒，顶级 SD 约 2 毛多/秒。
5. 也可在「设置」接入自己的中转站 API、配置提示词模板、同步/异步任务时间等。新版本可**一键更新**。

## 四、核心节点与功能详解

### 1. 视频抽帧 / 关键帧提取节点

- 把一段完整视频瞬间拆解为代表性分镜关键帧，可连线到生图节点做图生图、风格迁移，甚至排版九宫格。
- **三种模式**：
  - 标准模式：均分为 9 帧。
  - 间隔模式：按时间间隔抽帧（如 15 秒视频 → 15 帧）。
  - 智能场景检测：按画面切换敏感度抽帧，越不敏感帧数越少（4–6 帧）。
- **用途**：快速提取关键画面；拆解他人视频（抽帧 + 语音转文字 + AI 分析）；大视频不便直传大模型时，先抽核心帧再处理。
- **流转**：帧可单张/全部复制，或直接连文本节点问「视频讲什么」。

### 2. 文字拼接节点

- 把多段文字/变量拼成完整文本，支持换行、逗号、大括号 `{}` 连接。
- 场景：合并角色故事；在 ComfyUI 里把 API 返回的文件名拼上 IP 地址组成完整图片链接。

### 3. 网址转图片节点

- 把图片/视频的网址直接显示为图片，可作生图参考；视频同理，甚至可做九宫格切分。核心宗旨：让文本/音频/视频素材互相流转。

### 4. 九宫格拼图

- 旧版「反人类」，新版本支持从文件夹直接拖入文件，切分/拼图所见即所得，切完可继续连线。v1.2.10 起图片切分与拼图完全改版。

### 5. 万能节点（画布的灵魂功能）

- **定位**：猫猫AI画布的**灵魂功能**。无需写代码，仅通过配置 URL 和参数，就能把任何第三方 AI API 接入画布变成可视化工具。可对接 ComfyUI / RunningHub / 各类大模型 API，串成完整工作流。
- **使用步骤**：
  1. 画布右键添加「万能节点」。
  2. 点击节点右上角「齿轮 ⚙️」图标进入**配置模式**。
  3. 填入 API 请求地址（URL）、请求方法（Method）及必要的 Headers 和 Body（JSON 格式）。
  4. **【核心】定义动态变量**：在 Body 或 URL 中用双大括号 `{{变量名}}` 定义输入项，系统自动解析并在节点面板生成对应输入框/上传按钮。
- **变量支持规则**（按变量名前缀判断渲染控件类型）：
  - 无特殊前缀 → 文本输入框
  - 用 `|` 分隔选项 → 下拉选择框
  - 以 `image` 开头 → 图片上传控件
- 提示词本身支持文本、数组、JSON 格式。
- 注意：经万能节点的结果默认每 4.5 秒刷新一次（非实时，省资源），比直连略慢几秒。
- 参考节点（原文配图，未提取完整配置）：RunningHub 上传图片、TTS（openai 接口）。

### 6. 🌐 360° 全景图功能

- **简介**：内置「全景图节点」，将生成的 2D 全景长图无缝转换为沉浸式 3D 空间，可自由漫游、添加物体标注，并从任意视角截图，二次生成图像/视频。
- **核心能力**：
  1. 沉浸式 3D 漫游：支持**球状（Sphere）**和**柱状（Cylinder）**两种投影，鼠标拖拽无死角查看。
  2. 空间物体标注与放置：在任意位置添加自定义文本标签或几何占位符（可调颜色和大小）。
  3. AI 模型接入联动：结合大模型，按场景语义自动生成符合环境的 3D 模型并植入。
  4. 自由视角截图：满意视角下一键「截图并生成节点」，提取局部画面进入下一轮创作。
- **使用场景**：室内设计与看房（放置沙发/床占位符评估比例后截图重绘）；游戏场景开发（天空盒预览、转 2D 概念图）；虚拟展厅搭建（文本标注展品）；视频分镜分发（截不同机位生成连贯视频）。
- **使用步骤**：
  1. 用生图节点生成宽高比极长的全景图，右键添加 360 全景节点，连接生图输出→全景节点输入，自动渲染为 3D 预览。
  2. 悬浮节点点全屏图标进入沉浸模式；滚轮缩放，按住左键 360° 旋转。
  3. 全屏模式下控制台默认展开，输入物体名称、选颜色、调大小后添加，物体掉入 3D 空间；左侧浮动工具栏可位移/缩放。
  4. 调好视角点「截图并生成节点」，系统捕捉画面、退出全屏并生成新图片节点，可连文本/视频节点继续创作。

### 7. ☁️ 七牛云（对象存储）配置与使用

- **为什么需要**：许多第三方大模型 API（尤其高级视频模型）只接受公网 URL，不支持 Base64/本地路径。配置七牛云后，画布会在节点执行前自动把本地素材上传云端转为公网直链，桥接各类 API。
- **核心场景**：
  - **对接万能节点（Custom Node）**：接口要求 `image_url` / `video_url` 时，连好七牛云即可全自动「本地资源 → 云端上传 → 获取直链 → 注入请求」。
  - **对接 SD2 视频节点**：官方接口严格要求公网有效图片 URL，未配置会被拒（400/网络不可达）。
  - **减小存储负担**：大文件转七牛云链接，减轻浏览器 IndexedDB 压力，防卡顿崩溃。
- **配置步骤**：
  1. 准备：实名认证的七牛云账号，创建「公开读」空间。
  2. 获取密钥：控制台 → 密钥管理，复制 AccessKey / SecretKey。
  3. 获取空间名和外链域名：对象存储 → 空间管理 → 公开空间，记下空间名称与外链域名。
  4. 获取上传区域 Endpoint：依机房而定，如华东为 `https://upload.qiniup.com`。
  5. 在画布「设置 → 数据管理」标签页的云存储配置模块填入上述信息。

### 8. 数据备份与导出

- 设置内可将全部内容（项目、资源、多开环境、API/大模型配置）**一键打包**，用于本机备份、跨设备继续、或发给信任的同事协作（含敏感账号/KEY，需谨慎）。

## 五、一键发送资源到剪映

> 解决「素材生成出来了，但剪辑流程没打通」的痛点——无需先下载再导入（v1.3.3 起正式支持）。

1. 画布中生成图片/视频后，点该素材**顶部工具栏的「剪映」图标**，素材自动传入剪映。
2. **批量发送**：多张图片先「合并为图片盒子」得简略视图，再批量发送。
3. **外部图片注意**：从其他画布或网上复制来的图片，直接粘贴后点剪映图标可能**无效**——需先点左下角**「魔法棒」**转为**路径图片**，再走上述流程。
