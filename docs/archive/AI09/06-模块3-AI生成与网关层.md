## 3. 关键数据流

```
Jn() (L32490)
  → 选端点 I (L32796) → POST 网关 /v1/images|videos/generations (main.py L591/L641)
  → LovartClient 创建异步任务 → task_id
  → 轮询 pollUrl (L33005) → GET /v1/tasks/{id} (main.py L873)
      · 退避：pollInterval = (ie||3)*1e3（≥3s），退避 15s
      · deadline = Date.now()+9e5（15min），超时抛 '生成超时（15分钟）'（L32997/L33521/L34193）
      · AbortController 中断（L32993/L33518/L34190）
      · confirm：若 pending_confirmation 且 AUTO_CONFIRM=false，前端发 confirm（L32963）→ 网关 L882
  → 解析 images[].url（数组 [0]）
  → CDN? ii() 下载落盘 (L1888) → uploads/tasks/ → 绝对路径
  → 更新节点 data.imageUrl/videoUrl
  → dispatchEvent('mutiwindow-task-completed') (L38481 / L44406) → 监听 L31428 → Ev() rescan → 资源刷新
```
