## 1. 运转图景

画布是 V1 的核心 UI（ReactFlow @xyflow/react v12）。交互 handler 集中在一个巨型画布组件内（混淆名 `Fr`/`Ir`/`Lr`/`xn`/`Cn`/`wn`/`kn` 等均为该组件内 `useCallback`）。拖入文件 → `ii()` 落盘 18080 → `Z()` 创建对应类型节点。3D 导演台是特殊节点 `director3dNode`，组件 `Th`。

## 3. 关键数据流（带 file:line）

### 3.1 画布拖放（onDrop → 落盘 → 建节点）
```
Lr(e)  @L36293
  ├─ 模板拖入：getData('application/x-yimao-template') → JSON.parse → sn.current(e,t) 导入子图  @L36301-36304
  ├─ 任务拖入：getData('application/x-mutiwindow-task') → 按 mediaType 建节点：
  │     video → Z('discountVideoNode', {videoUrl})  @L36318
  │     audio → Z('audioPlayerNode', {audioUrl})    @L36322
  │     text  → Z('textNode', {text})               @L36325
  │     image → Z('imageNode', {imageUrl})          @L36328
  └─ 文件拖入：dataTransfer.files → 按扩展名判类型 →
        text → FileReader → Z('textNode')           @L36351-36359
        其它 → ii(e,{subfolder:'canvas/drop'})  @L36363  →  落盘 18080
              → Z('imageNode'|'audioPlayerNode', {url})  @L36370-36378
```
> 落盘路径 `uploads/canvas/drop/`（与 ARCHITECTURE L2.5 一致）；拖入即走 `ii()` 落盘，符合本地模式闭环。

### 3.2 连线交互
```
Fr(e,t)  @L36245  (onConnectEnd)
  └─ 若 t.isValid 为假但有 fromNode/fromHandle → 计算落点 → 建 ghost-target + ghost-edge 预览 → ve({type:'connection'}) 打开连接菜单
xn(e)    @L31883  (onConnect) → 标准边创建
```

### 3.3 右键菜单
```
Cn(e)  @L31933  (onPaneContextMenu) → 画布空白右键（添加节点/粘贴等）
wn(e,t)@L31947  (onNodeContextMenu) → 节点右键（删除/复制/属性）
kn(e,t)@L31980  (onNodeClick)       → 节点选中/打开
```

### 3.4 3D 导演台接入
- 节点类型：`director3dNode`（注册 @L31141 `director3dNode: Th`；UI 入口 @L35724 `Z('director3dNode',...)`、标签「3D导演台」@L35725）
- 渲染体：`Director3D` 组件（@L28289+，含 `director3d-overlay`/`topbar`/`body` 结构）
- 捕获：`Kn`@L32367（onCaptureToBox）供 `director3dNode`/`panoramaNode` 截图入 imageBox（@L35599）
