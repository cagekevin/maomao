## 1. 运转图景

系统启动时（`src/main.tsx` lazy 加载 `App.js`），前端完成：
1. **配置层加载**：`config.js` 集中导出所有端点/开关（本地引擎 :18080、网关 :9004、USE_LOCAL_ENGINE 等）。
2. **鉴权降级**：本地模式下去登录，`Oa()` 恒返回本地 token，使所有 `getCurrentToken` 拦截放行。
3. **存储管理器选型**：`Q`（混淆名 StorageManager）按可用性选择后端（localTool / chrome.storage / localStorage / localforage）。
4. **URL 拼接助手**：`tr`/`nr`/`ar`/`cr` 统一拼接 `/api` 前缀与 base URL。
5. **画布状态 key 生成**：`Cr` 生成 `canvas-state-v1-{projectId}` 类的持久化键。

> 本模块是其余模块的地基：模块2/3/4 的所有 KV/文件/资源调用都经由 `Q` + 存储后端；AI 请求地址由 `config.js` + `tr`/`ar` 派生。

## 3. 关键数据流（带 file:line）

### 3.1 启动初始化链路
```
src/main.tsx
  → React.lazy(() => import('./_engine/App.js'))        // 入口壳
  → bootstrap 内：Q 初始化 → 检查 window.localTool 连接状态
  → config.js 全量导出被 App.js 直接 import 引用
  → Oa() 恒返回 local-mode-token → Jh() 构造 Bearer 头
```

### 3.2 鉴权降级（去登录）
- `Oa()` @L3543：`localStorage.getItem(Ca /*'auth_token'*/) || 'local-mode-token'`
- `Ba()` @L3637：`!!Oa()` → 永远 `true`（本地模式）
- `ka()` @L3547 写 token；`Aa()` @L3552 清 token（"退出登录"→"重置配置"文案在 `Zr`@L43893）

### 3.3 配置层加载（`config.js`）
全量导出（已读原文）：
- `LOCAL_ENGINE` = `{host:'127.0.0.1', port:18080, base}` （L12–18）
- `ENDPOINTS` = `[{label:'API网关', url:'http://127.0.0.1:9004'}]` （L25–27）
- `DEFAULT_ENDPOINT` = `'http://127.0.0.1:9004'` （L30）
- `USE_LOCAL_ENGINE` = `true` （L36）
- `REMOTE_BASE` = `'http://127.0.0.1:9004'` （L39）
- `localEngineBase()` / `getLocalEngineBase` （L42–47）
- `GAS_CLOUD_SYNC_URL` （L75，云端同步按钮）

> 端口坑：网关实际连 9004（见 PROJECT_ORIGIN §4）；config 与 App.js `ot='http://127.0.0.1:9004'` 一致。

### 3.4 存储后端选型（`Q` → wr/Mr/Nr/Pr）
- 主后端 `wr`（localTool）经 `window.localTool.getKV/saveKV`（L1301/L1310），`isAvailable()` 看 `window.localTool?.status?.isConnected`（L1332）。
- 兜底链：`Mr`(chrome.storage) → `Nr`(localStorage) → `Pr`(localforage)。
- `jr`(@L1345) 对资源类 key（`img_`/`img_thumb_`/`video_thumb_`）加 10min TTL 内存缓存 + 并发锁，降低重复读取。

### 3.5 画布状态持久化键
- `Cr(e)` @L1294 = `${Z.CANVAS_STATE_PREFIX}${e}`；`Pr`(localforage) 落盘（画布 onChange → `canvas-state-change` 事件 → `Pr.set`）。
- 见 ARCHITECTURE X2 事件总线 `canvas-state-change`。
