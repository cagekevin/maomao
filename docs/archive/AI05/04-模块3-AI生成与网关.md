# 模块3 · AI 生成与网关层（AI05 / T2.3）

> 边界：前端 ↔ apimart-gateway(:9004) ↔ Lovart，异步 task 轮询 + 结果落盘 18080。
> 锚点：T0.1 坐实的"真实 AI 生成派发函数" = `Jn`(L32490) 内部轮询分支（非独立函数）。

---

## 1. 运转图景

生图/生视频由 `Jn`(L32490, imageNode 生图主回调) 派发：
1. 收集上游 image/video/text 上下文 → 构造 prompt（`g()` 清洗 ≤6000 字）。
2. 调网关 `POST /v1/images|videos/generations`。
3. 响应含 `task_id` → 进入**异步轮询分支**（L32987-33071）。
4. 轮询 `GET /v1/tasks/{id}`（≥3s，429 退避 15s，15min deadline）。
5. `status==='completed'` → 提取 `images[].url[0]`（注意是数组[0]）。
6. **落盘**：`ii(u,{subfolder:'tasks',...})` → `POST /api/files/upload` → 18080（红线 §3.2：严禁 CDN 直链）。
7. 回填节点 `data.imageUrl`；发 `mutiwindow-task-completed` → `Ev()` rescan → 资源面板刷新。

## 3. 关键数据流（端到端，带边界）

```
Jn(e, prompt, size, model) @L32490
  → 收集上游上下文(gs/hs) + 清洗 g() @L32549
  → POST gateway /v1/images|videos/generations   (网关 main.py L591/L641)
  → 响应 t
  ├─ 有 task_id (L32988) → 异步轮询分支:
  │    while(true):
  │      GET `${R}/v1/tasks/${taskId}`  (R=网关9004, L33005)
  │      status==='completed' (L33028):
  │        imgUrl = result.images[0].url[0]  (数组[0]! L33031)
  │        ii(imgUrl,{subfolder:'tasks'}) → POST /api/files/upload → 18080  (L33049)
  │        u = persisted.url; 回填节点 data.imageUrl
  │        dispatch 'mutiwindow-task-completed' → Ev() rescan → 资源刷新
  │      status==='failed' → throw
  │      pending_confirmation → markNeedsConfirm 退出轮询 (L33061)
  │      429 → backoff×1.5 上限15s; else 间隔3s (L33067-69)
  │      deadline 15min (L32997)
  └─ 无 task_id → 同步解析 (L33076)
```
