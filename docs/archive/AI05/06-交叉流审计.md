## X3. Rescan 孤儿清理与资源入库一致性（缝合 模块2）

- `Ev()`(@L42883) → localTool rescan：扫 `uploadDir` 子目录，孤儿清理**只清 `source='local-tool'`**（TASKS P2 #7）。

## X4. 配置加载与双服务 base URL 桥接（缝合 模块1）

- `USE_LOCAL_ENGINE=true` → `localEngineBase()`=18080（localTool）；`false` → `REMOTE_BASE`(9004)。
- `Hr`(var L117)/`vv`(var L118)/`U_`(var L233) 三处硬编码 `127.0.0.1:18080`，值恒等。
- 真风险：`USE_LOCAL_ENGINE=false` 时文件请求走 9004 无上传路由（ARCHITECTURE X3.3）→ 报错。属 TASKS P2"统一 base URL"。
