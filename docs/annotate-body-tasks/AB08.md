# AB08 — 函数体段落注释（3 个节点）

> 在节点组件函数体内添加分段注释，不给代码动手术，只加记号。

## 规则

1. 找到函数体（grep 变量名或 `Y.memo`) → 读完整函数
2. 在逻辑段落间插入 `// ── 段落名 ──` 注释
3. **不改任何代码**，只加注释行
4. 段落分类参考：状态声明 / effect / 回调函数 / 渲染 / 工具函数

## 示例

```js
var ImageNodeComp = Y.memo(({ id, data, selected }) => {
  // ── 状态 ──
  let [loading, setLoading] = Y.useState(false);
  let imgRef = Y.useRef(null);
  // ── 上传处理 ──
  let handleUpload = Y.useCallback((e) => { ... }, []);
  // ── 右键菜单 ──
  let handleContextMenu = Y.useCallback((e) => { ... }, []);
  // ── 渲染 ──
  return X.jsxs("div", { ... });
});
```
