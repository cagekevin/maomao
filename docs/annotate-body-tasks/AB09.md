# AB09 — 节点函数体重写（3 个）

> 🚫 **禁止直接修改 src/App.js！** 只在本文件代码块中提交重写结果。运行 `node scripts/merge-rewrites.cjs` 统一写入。
> ⚠️ 铁律：改写后功能必须和原来完全一样。只改局部变量名和加注释，不改任何逻辑。

## 规则
1. 不改函数签名和模块级变量名
2. 重命名函数体内局部变量：a,b,c→loading,setLoading
3. 加段落注释：`// ── 状态 ──` `// ── 回调 ──` `// ── 渲染 ──`
4. 保持 vendor/React 用法不变：不动 Y.useState、X.jsx 等
5. 不改 JSX 结构

## 提交格式（重要！）
```
| stickyNoteNode | StickyNoteNodeComp |
```js
// ── 状态 ──
const [loading, setLoading] = Y.useState(false);
// ...
```
|
```
