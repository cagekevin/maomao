# AB05 — 节点函数体重写（3 个）

> 🚫 **禁止直接修改 src/App.js！** 只在本文件代码块中提交重写结果。运行 `node scripts/merge-rewrites.cjs` 统一写入。
> ⚠️ 铁律：改写后功能必须和原来完全一样。只改局部变量名和加注释，不改任何逻辑。

## 规则
1. 不改函数签名和模块级变量名
2. 重命名函数体内局部变量：a,b,c→loading,setLoading（本任务中即顶层解构别名 updateNodeData/setNodes/setEdges/getNode 等，以及含义确定的赋值）
3. 加段落注释：`// ── 状态 ──` `// ── 回调 ──` `// ── 渲染 ──`
4. 保持 vendor/React 用法不变：不动 Y.useState、X.jsx 等
5. 不改 JSX 结构

## 提交格式（重要！）
每个节点一块：先用一行 `| 节点类型 | 变量名 |`，紧接着写 js 代码块（代码块结束后再跟一行 `|`）。

## 任务
1. **rhWebappNode** (`RhWebappNodeComp`)
2. **videoExtractNode** (`VideoExtractNodeComp`)
3. **videoToGifNode** (`VideoToGifNodeComp`)
