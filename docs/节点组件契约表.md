# 节点组件契约表

### imageNode (`li`)
| 属性 | 值 |
|------|----|
| props | `id`, `data{imageUrl, imageUrlRef, thumbnailUrl, imageAvailable, label, _styleWidth, onZoom, onCrop, onEdit, onSendToActiveTab, onShowToast}`, `selected`, `width` |
| callbacks | `onZoom(id, ref, url)`, `onCrop(id, url, ref)`, `onEdit(id, ref, url)`, `onSendToActiveTab(url)`, `onShowToast(msg)` |
| output | `imageUrl`, `thumbnailUrl`, `imageUrlRef`, `imageAvailable`, `label`（经 `updateNodeData` 回写） |
| AI生成 | 否 → 纯展示/上传节点；`onChange` 本地上传 → `updateNodeData` 回写 `imageUrl` |
| 上游连接 | `target`(Left) 接收 `imageUrl` |
| 下游连接 | `source`(Right) 输出 `imageUrl` → `promptNode`/`gridSplitNode`/`cropNode`/`gridMergeNode` |

### promptNode (`Ya`)
| 属性 | 值 |
|------|----|
| props | `id`, `data{prompt, aspectRatio, imageSize, selectedModel, apiFormat, selectedContextResources, expanded, drawingModel, imageUrl, imageUrlRef, imageUrlThumbRef, thumbnailUrl, loading, errorMessage, onZoom, onCrop, onEdit, onSendToActiveTab, onShowToast, onAddImage, onGenerate, onStop, inputWidth, inputHeight, presetPrompts}`, `selected`, `width` |
| callbacks | `onGenerate(id, prompt, '1024x1024', model, apiFormat, batch)`, `onStop(id)`, `onZoom`, `onCrop`, `onEdit`, `onSendToActiveTab`, `onShowToast`, `onAddImage` |
| output | `prompt`, `aspectRatio`, `imageSize`, `selectedModel`, `apiFormat`, `selectedContextResources`, `expanded`, `inputWidth`, `inputHeight`；`imageUrl`/`imageUrlRef`/`thumbnailUrl`/`loading`/`errorMessage` 由父组件生成阶段回写 |
| AI生成 | 是 → `onGenerate` → 网关 `http://127.0.0.1:9004` → 回写 `data.imageUrl`/`imageUrlRef`/`thumbnailUrl` |
| 上游连接 | `target`(Left) 接收 `imageUrl`/`text`（来自 `imageNode`/`imageBoxNode`/`videoNode`/`videoExtractNode`/`textNode`/`urlToImageNode`/`fileToUrlNode`/`panoramaNode`） |
| 下游连接 | `source`(Right) 输出 `imageUrl` → `gridSplitNode`/`cropNode`/`gridMergeNode` |

### cropNode (`eo`)

### audioPlayerNode (`ps`)

|------|----|

### customNode (`ms`)
| 属性 | 值 |
|------|----|
| props | `{id, data, selected}`，data：`config{apiUrl,method,headers,body,outputType,executionMode,resultPath,taskIdPath,polling*,rawTextOutput,variableFormats}, configMode, resultData, loading, progress, errorMessage, label` |
| callbacks | `onAIAssist(prompt, config)`(AI 辅助生成配置)、`onSaveTemplate(name, config)`(保存为模板)、`onGenerateCustom(id)`(开始处理)、`onStop(id)`、文件上传 `x`(变量素材)、完成配置 `v()` 写回 `config/configMode` |

### urlToImageNode (`Wc`)
