# 常见报错速查

> 遇到报错 → 搜关键词 → 秒定位根因。不用从零排查。

## 重命名相关

| 报错 | 根因 | 解决 |
|------|------|------|
| `CustomNodeComp is not defined` | 逗号链声明 `ms = Y.memo(` 无 `var` 前缀，sed 没匹配到，只换了 lg 映射没换声明 | 检查 App.js 中变量声明是否一致，`grep <新名> src/App.js` 确认声明+引用都替换了 |
| `ImageNodeComp is not defined` | 同上，`li = Y.memo(` 声明漏换 | 同上 |
| `Cannot access 'Dj' before initialization` | 重命名导致 ES module 变量初始化顺序改变，触发了 TDZ | `git checkout src/App.js` 回退，`npm run build && node scripts/check-build.cjs` 验证 |

## 模板相关

| 报错 | 根因 | 解决 |
|------|------|------|
| 模板面板打不开 | `local_templates` KV 数据格式异常（如 graphData.nodes 为空） | `node -e "..."` 清空 local_templates 键，或用 seed-templates.cjs 重新播种 |
| 模板数据消失 | localTool 非优雅退出（被杀进程），内存未落盘 | 正常 Ctrl+C 退出 localTool，确保 SIGINT→shutdown→saveDb |

## 拆解相关

| 报错 | 根因 | 解决 |
|------|------|------|
| `Bd is not a function` | 从 App.js 抽取函数改变了 ES module 加载图拓扑，vendor 内部变量初始化失败 | 回退，不拆共享函数（翻车点 #16） |
| `xxx is not a function` (刚拆完) | 新文件 import 了 App.js，产生循环依赖 | 不 import App.js，用事件总线（翻车点 #6） |
| `npm run build` 通过但 Chrome 打不开 | 构建产物语法正确但运行时初始化顺序错误 | `node scripts/check-build.cjs` 检查 TDZ 风险 |

## 网关/存储

| 报错 | 根因 | 解决 |
|------|------|------|
| `404` 从 `:9004` | 未实现的 API 端点，属已知噪音 | 忽略，CLAUDE.md 已标注 |
| `RootErrorBoundary` null 异常 | React 错误边界噪音 | 忽略 |
| `Q.setObject` 返回 false / 写入失败 | localTool 未启动或断开 | `curl http://127.0.0.1:18080/api/status` 检查，重启 localTool |
| 数据保存后丢失 | localTool 未优雅退出 | 用 Ctrl+C 退出 localTool（不要杀进程/关终端窗口） |

## 构建

| 报错 | 解决 |
|------|------|
| `sh: cross-env: command not found` | `npm install`（`node_modules` 未安装） |
| `invalid JS syntax` | 检查 App.js 语法：`node -e "require('acorn').Parser.parse(fs.readFileSync('src/App.js','utf-8'),{ecmaVersion:'latest',sourceType:'module'})"` |
| vendor 导出找不到 | App.js import 行第 3 行引用了不存在的 vendor 导出 → 检查 vendor 导出名是否变动 |
