# VA02 — vendor.js 导出标注任务（40 个）

> vendor.js 中 `内部名 as 导出名` 格式的 195 个导出，标注每个导出对应的真实 API。

## 方法

1. 对每个导出名，grep App.js 看它怎么被 import：
   ```
   import { Nr as le } from "./vendor/vendor.js"
   # Nr 在 App.js 中别名 le → grep "\ble\b" 看使用场景
   ```
2. 根据使用模式判断对应哪个库的哪个 API：
   - 如果是 JSX 工厂 → React.createElement
   - 如果是组件包装 → React.memo / forwardRef
   - 如果是 hook → useState / useEffect / useRef...
   - 如果是图标 → lucide-react 的具体图标名
   - 如果是工具 → lodash 的具体函数
3. 不确定标注 `(?)`

## 输出格式

```
导出名 = 库::API # 使用场景
Nr = React::createPortal # Un = Nr(), createPortal 调用
Ar = React::createElement # X = Ar(), jsx 工厂
Mr = React::memo # Y.memo(...) 包组件
Wt = lucide::X # 关闭按钮图标
```
