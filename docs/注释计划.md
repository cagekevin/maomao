# App.js 添加注释计划

> 目的：在关键代码段上方添加 `// @section` 注释，降低 IDE 跳转和阅读成本。
> 另一个 AI 只需按行号插入，不需要理解代码逻辑。

---

## 策略

在每个模块级函数/组件定义的上一行插入 `// @section: 名称 (功能简述) @L行号`。
不修改任何代码，只加注释。

## 执行方式

给另一个 AI 的 prompt：
```
在 src/App.js 中，在指定行号上方插入一行注释。
每行注释以 // 开头，紧挨目标行的上一行插入。
不修改任何代码，只插入注释行。

示例：在 L1706 上方插入 "// @section: PromptNode(Ya) 生图节点 — prompt输入/模型选择/生成按钮/停止按钮 @L1706"
```
