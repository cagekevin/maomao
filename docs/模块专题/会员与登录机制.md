## 一、这是什么

maomao 有两条并存的"身份/权益"通路：

1. **官方账号登录**（在线模式）：手机号+验证码 / 账号密码 → 拿到 `token` → 拉取会员等级、余额、模型权益。
2. **本地模式（local-mode）**：免登录，直接注入一个 `UNLIMITED` 本地会员，不请求官方 `/user/info`。

当前构建 `USE_LOCAL_ENGINE` / `LOCAL_MODE_ALLOW_ALL = true`（`src/config.js` L90），默认走本地模式，但官方账号登录的完整代码路径仍保留可用。

```text

export const LOCAL_MODE_ALLOW_ALL = true;
```

- 在线模式会员过期：`membership` 键被校验删掉，回落 `FREE`（§四），模型权益按官方 `entitlements` 重新收紧。

## 八、与 GAS 云同步机制的衔接（重点）

衔接点在 **数据层面**，而非 **前置条件层面**：

1. **GAS 云同步不要求官方登录。** `syncToCloud()` 直接 POST 到 `GAS_CLOUD_SYNC_URL`，不带 `Bearer` token，也不检查 `Ne`。它是独立的可公开访问的 Google Apps Script 端点（见 `GAS云同步机制.md` §二）。

```text
src/App.js:41365-41369（syncToCloud 推送键集合）
syncToCloud = async () => {
  ...
  for (let t of [`app_settings`, `api_configs`, `users`, `membership`,
                 `projects`, `presetPrompts`, `customNodeTemplates`,
                 `modelSchedules`, `cloud_storage_config`, `local_templates`]) {
```

2. **登录/会员数据被 GAS 一起同步。** 上面列表里的 `users` 与 `membership` 正是本专题的账号与会员态。也就是说：在线模式登录后产生的会员等级、用户列表，会随 GAS 推送上云、随 GAS 拉取回本地——实现跨设备会员态同步；本地模式的 `UNLIMITED` 会员态同样会被推上去（但那只是本地标记，换设备拉回仍是 `UNLIMITED`）。

3. **结论：**
   - **登录不是 GAS 云同步的前置条件**——未登录也能推/拉云端（此时 `users`/`membership` 是默认/空值）。
   - **但 GAS 是会员态的同步通道之一**：官方登录拿到的会员数据，通过 GAS push 落到云端，pull 时写回 `membership` 键，再经 §四过期校验恢复会员态。
   - 两条通路解耦：官方登录管"权益怎么来"，GAS 管"权益/配置怎么跨设备搬"。

> 详见同目录 `GAS云同步机制.md`（push 键集合 L41369、pull 合并语义）。改动 GAS 推送键集合时，务必保留 `users` 与 `membership`，否则会员态无法跨设备同步。

## 九、改码前必查

1. **token 键名三处必须一致**：`storageKeys.js` 的 `AUTH_TOKEN`(`auth_token`)、`config.js` 的 `AUTH_TOKEN_KEY`(`auth_token`)、`constants.js` 的 `Ca`(`auth_token`) 必须同字符串，否则 `Oa()/ka()/Aa()` 读写错位。
2. **401 自动登出**（`App.js` L36143）依赖 `Aa()` 真正清掉 `auth_token` 与存储层；若改 token 存储位置，必须同步改 `Oa/ka/Aa`，否则"登录态永远不清"。
3. **会员过期校验**（`App.js` L40586-40593）是 reload 后恢复会员态的唯一闸门，删掉会导致过期会员仍显示有效。
4. **本地模式兜底**：`LOCAL_MODE_ALLOW_ALL = true` 时 `modelEntitlements.js` 全模型放行；若要恢复"会员限制模型"，需先关掉该开关并接好 `/user/model-entitlements`。
5. **GAS 推送键集合**（`App.js` L41369）含 `users`/`membership`，改 GAS 同步逻辑时不要误删，否则破坏会员态跨设备同步（见 §八）。
