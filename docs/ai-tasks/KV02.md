# KV02 — KV 存储键读写面提取（5 个）：

> 标注每个 KV 键的读写位置和数据格式。

## 方法
1. grep 键名在 App.js 中的出现（注意 storageKeys.js 中的常量名）
2. Q.setObject(key, ...) → 写入口
3. Q.getObject(key) → 读入口
4. grep syncToCloud 键集合 → 是否进 GAS 同步

## 输出格式

```
presetPrompts:
  写: promptManager.js saveAndNotify()
  读: PromptLibrary.jsx mapToLibraryCards()
  GAS: 是
  结构: [{title,type,prompt,enabled},...]
```
