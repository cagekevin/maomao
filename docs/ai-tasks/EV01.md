# EV01 — 事件总线字典提取（6 个）

> 标注每个 CustomEvent 的派发方和消费方。

## 方法
1. grep 事件名在 App.js + src/ 其他文件中的出现
2. 找到 dispatchEvent → 派发方
3. 找到 addEventListener → 消费方
4. 说明事件携带的 detail 字段

## 输出格式

```
open-shortcuts-modal:
  派发: 设置菜单按钮 onClick
  消费: App.js → Le(true) 打开快捷键面板
  detail: 无
```

### `builtin-panel-switch-schedule` (window CustomEvent)
| 属性 | 值 |
|------|----|
| 派发 | `App.js`@L41161 — 处理函数 `t`（注册在 `mutiwindow-open-schedule-settings` 监听内）：先 `Te('settings')`+`De('builtin')` 切到内置设置面板，再 `setTimeout(50ms)` 派发。即多窗口“打开调度设置”信号到达时触发 |
| 消费 | `App.js`@L38500 — `BuiltinModelPanel`(N_) 内 `useEffect` 监听，回调 `() => t('schedule')`，把内置面板切到 schedule（模型调度）视图 |
| detail | 无（`new CustomEvent('builtin-panel-switch-schedule')` 无第二参） |

### `click` (原生 DOM 事件，非 CustomEvent)
| 属性 | 值 |
|------|----|
| 派发 | 浏览器原生派发（代码内无 `window.dispatchEvent` 调用） |
| 消费 | `App.js`@L5229 — 回调 `() => N(null)`：依赖 `M` 状态的浮层监听，点击空白处取消选中/关闭；`App.js`@L8327/@L8491/@L10352 — `document.addEventListener('click', n)`（延迟100ms注册）下拉/弹层“点击外部关闭”处理器 `n`；`App.js`@L26826 — `window.addEventListener('click', e, true)`（捕获阶段，配合 mousedown）点击外部检测关闭 |
| detail | 原生 `MouseEvent`，无自定义 detail |

### `contextmenu` (原生 DOM 事件，非 CustomEvent)
| 属性 | 值 |
|------|----|
| 派发 | 浏览器原生派发（右键时） |
| 消费 | `App.js`@L5229 — 与 `click` 同一处理器 `() => N(null)`：右键空白处时取消选中/关闭浮层 |
| detail | 原生 `MouseEvent`（含鼠标坐标），无自定义 detail |

### `error` (原生 DOM 事件，非 CustomEvent)
| 属性 | 值 |
|------|----|
| 派发 | `FileReader` 读取失败时原生派发 |
| 消费 | `App.js`@L24378 — `r.addEventListener('error', () => n(r.error ?? Error('模型文件读取失败')))`：模型文件读取失败 → reject Promise |
| detail | 原生事件，无自定义 detail |

### `export-project` (window CustomEvent)
| 属性 | 值 |
|------|----|
| 派发 | `App.js`@L42148 — 项目菜单“导出项目”按钮 `onClick`：`window.dispatchEvent(new CustomEvent('export-project'))` |
| 消费 | `src/components/panels/ImportExportPanel.js`@L23 — `window.addEventListener('export-project', t)`，回调 `t = () => r.current()`（`r` 为导出数据 ref，触发执行导出） |
| detail | 无 |

### `import-project` (window CustomEvent)
| 属性 | 值 |
|------|----|
| 派发 | `App.js`@L42142 — 项目菜单“导入项目”按钮 `onClick`：`window.dispatchEvent(new CustomEvent('import-project'))` |
| 消费 | `src/components/panels/ImportExportPanel.js`@L23 — `window.addEventListener('import-project', e)`，回调 `e` 创建隐藏 `<input type="file" accept=".json">` 并 `e.click()` 打开文件选择器，选中后调用 `n.current(e)` |
| detail | 无 |
