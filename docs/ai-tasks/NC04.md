# NC04 — 节点组件契约提取（6 个）

> 提取每个节点的 props、回调、产出 data 字段、上下游连接关系。

## 方法
1. 在 App.js 中找到对应函数定义（grep variable name）
2. 读取函数参数（prop 解构）→ 得到 props
3. 搜索 onClick/onChange/onSubmit → 得到 callbacks
4. 搜索 W(setNodes) 或 updateNodeData → 得到 output 字段
5. 搜索 onAddImage/onPushImagesToImageBox 等 → 判断是否 AI 生成类

## 输出格式

```
imageNode (li):
  props: {id, data{imageUrl,label,thumbnailUrl}, selected}
  callbacks: onCrop(), onZoom(), onEdit(), onSendToActiveTab()
  output: {imageUrl, thumbnailUrl, loading, errorMsg}
  AI生成: 是 → handleImageGenerate → :9004 → 回写 imageUrl
  connected: 接收 promptNode/urlToImageNode → 输出到 gridSplitNode
```

### textConcatNode (`Rc`)
| 属性 | 值 |
|------|----|
| props | `{id, data{separator, prefix, suffix, text}, selected}` |
| callbacks | 内联：前缀 input onChange → `updateNodeData({prefix})`；分隔符 input onChange → `{separator}`；后缀 input onChange → `{suffix}`；`useEffect` 自动把拼接结果写回 `text` |
| output | `{text, separator, prefix, suffix}`（text 为按 前缀+各输入.join(分隔符)+后缀 拼接结果） |
| AI生成 | 否（纯文本拼接） |
| 上游连接 | 文本源节点：`textNode` / `audioNode` / `textConcatNode`（target 默认句柄，左） |
| 下游连接 | 文本消费节点：source id=`text`（右）→ `promptNode`、`urlToImageNode` 等 |

### urlToImageNode (`Wc`)
| 属性 | 值 |
|------|----|
| props | `{id, data{inputUrl, imageUrl, lastFetchedUrl, onShowToast}, selected}` |
| callbacks | URL input onChange → `updateNodeData({inputUrl})`；重新获取按钮 onClick (`p`) → 拉取并转 base64；`useEffect` 监听 `inputUrl` 变化自动拉取（避免重复 fetch `lastFetchedUrl`）；图片双击 → `onZoom(imageUrl)` |
| output | `{inputUrl, imageUrl (base64), lastFetchedUrl}` |
| AI生成 | 否（通过 `zc` 拉取 URL → FileReader 转 base64） |
| 上游连接 | `textNode`（含 http 的文本）/ `imageBoxNode`（target 默认句柄，左）；也支持手动输入 URL |
| 下游连接 | 图片消费节点：source id=`image`（右）→ `imageNode`、`gridSplitNode`、`panoramaNode`、`director3dNode` 等 |

### fileToUrlNode (`qc`)
| 属性 | 值 |
|------|----|
| props | `{id, data{cloudStorageConfig, loading, errorMsg, resultUrl, onShowToast}, selected}` |
| callbacks | 生成链接按钮 onClick → 校验有输入+已配置对象存储 → `Kc` 上传到 S3 兼容云存储（`zc` PUT）→ 回写 `resultUrl`；复制按钮 → `navigator.clipboard` 复制 `resultUrl` |
| output | `{loading, errorMsg, resultUrl}`（resultUrl 为云端 URL；cloudStorageConfig 读取自 data 不回写） |
| AI生成 | 否（对象存储上传，非 AI） |
| 上游连接 | `imageNode`/`videoNode`/`audioNode`/`textNode`/`customNode`（读其 imageUrl/videoUrl/audioUrl/text/customResultData，target id=`file-input`，左） |
| 下游连接 | URL 消费节点：source id=`url-output`（右）→ `imageNode`(URL 字段)、`promptNode` 等 |

### panoramaNode (`Yc`)
| 属性 | 值 |
|------|----|
| props | `{id, data{panoType, highQuality, imageUrl, aspectRatio, label, onCaptureToBox, showToast}, selected}`（内部另用 `getNode`/`setNodes` 调整节点尺寸） |
| callbacks | 截图按钮 `F([0])`/`F([90,180,270,0])`/`F([0..330 每30°])` → `Jc.capture` 渲染多视角 → 回写 `imageUrl` 并 `onCaptureToBox` 推入 imageBox；滚轮 `onWheel` 调 fov(30~120)；比例 select onChange；全景类型 select onChange → `{panoType}`；高画质 checkbox onChange；全屏漫游按钮 `u(true)` |
| output | `{aspectRatio, highQuality, panoType, imageUrl}`（截图回写 imageUrl）；`onCaptureToBox(id, [{url,label}])` 推送到 imageBox |
| AI生成 | 否（Three.js 全景球/柱查看 + 离屏截图） |
| 上游连接 | 图片节点 imageUrl（优先取上游连入、且非 video/audio/text 的 `imageUrl`，否则用 data.imageUrl，target 默认句柄，左） |
| 下游连接 | 图片消费节点：source 默认句柄（右）→ `imageNode`、`gridSplitNode`、`director3dNode` 等 |

### director3dNode (`Th`)
| 属性 | 值 |
|------|----|
| props | `{id, data{directorProject, imageUrl, thumbnailUrl, syncedCaptureIds, onCaptureToBox}, selected}` |
| callbacks | 点击节点/打开按钮 → `s(true)` 弹出 `wh` 导演台（`initialProject`/`initialPanoramaUrl`）；`onExit=f` 回调 → 收集机位截图 `onCaptureToBox(id, captures)` → `updateNodeData({directorProject, syncedCaptureIds, imageUrl(thumbnailDataUrl)})`；缩略图经 `ii()` URL 化后回写 `imageUrl`/`thumbnailUrl` |
| output | `{directorProject, syncedCaptureIds, imageUrl, thumbnailUrl}`；`onCaptureToBox` 推送机位截图到 imageBox |
| AI生成 | 否（3D 导演台：机位截图 + 缩略图导出，无 AI 生成调用） |
| 上游连接 | `imageNode` / `imageBoxNode`（读 imageUrl，或 imageBox 的 activeIndex 图片，target 默认句柄，左） |
| 下游连接 | 图片消费节点：source 默认句柄（右）→ `imageNode`、`gridSplitNode`、`panoramaNode` 等 |

### imageBoxNode (`Ih`)
| 属性 | 值 |
|------|----|
| props | `{id, data{images, activeIndex, expanded, selectedIds, onShowToast, onZoom, onSendToActiveTab}, selected}`（`images:[{id,url,thumb,label,source,createdAt}]`） |
| callbacks | 本地上传 `z`/粘贴 `paste` 监听/拖拽 `B` → `O` 追加 images；从连线一键导入 `re`；删除单张 `k`；批量删除 `A`；切换选中 `T`；上一张 `N`/下一张 `P`；设为当前 `j`；重排 `M`；放大 `ae`→`onZoom`；发送到活动标签 `se`→`onSendToActiveTab`；发送到剪映 `V`(`qn`)，展开 `F` |
| output | `{images, activeIndex, expanded, selectedIds}`（纯图片容器，自身不产生新图） |
| AI生成 | 否（图片收纳盒，接收上游图片/上传/粘贴/拖拽） |
| 上游连接 | 图片源节点（target id=`in`，左）：`imageNode`/`imageBoxNode`/`videoExtractNode`/`gridSplitNode`/`gridMergeNode`/`cropNode`/`urlToImageNode`/`fileToUrlNode`/`panoramaNode` 等（读 imageUrl / extractedImages / images） |
| 下游连接 | 图片消费节点（source id=`active`，右）：`director3dNode`/`panoramaNode`/`imageCompressNode`/`faceMosaicNode`；同时作为 `onCaptureToBox`/`onPushImagesToImageBox` 的接收方 |
