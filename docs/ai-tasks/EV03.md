# EV03 — 事件总线字典提取（6 个）

> 标注每个 CustomEvent 的派发方和消费方。

## 方法
1. grep 事件名在 App.js + src/ 其他文件中的出现
2. 找到 dispatchEvent → 派发方
3. 找到 addEventListener → 消费方
4. 说明事件携带的 detail 字段

## 输出格式

```
open-shortcuts-modal:
  派发: 设置菜单按钮 onClick
  消费: App.js → Le(true) 打开快捷键面板
  detail: 无
```
