# NC01 — 节点组件契约提取（6 个）

> 提取每个节点的 props、回调、产出 data 字段、上下游连接关系。

## 方法
1. 在 App.js 中找到对应函数定义（grep variable name）
2. 读取函数参数（prop 解构）→ 得到 props
3. 搜索 onClick/onChange/onSubmit → 得到 callbacks
4. 搜索 W(setNodes) 或 updateNodeData → 得到 output 字段
5. 搜索 onAddImage/onPushImagesToImageBox 等 → 判断是否 AI 生成类

> 数据来源（解编译节点）：
> `docs/archive/reference/decompiled/nodes/{imageNode,promptNode,textNode,cropNode,gridSplitNode,gridMergeNode}.raw.js`
> 回调绑定来源：`src/App.js` 的节点 data 装配逻辑（行 33367-33400 附近）。

## 输出格式

```text
imageNode (li):
  props: {id, data{imageUrl,imageUrlRef,thumbnailUrl,imageAvailable,label,_styleWidth,onZoom,onCrop,onEdit,onSendToActiveTab,onShowToast}, selected, width}
  callbacks: onZoom(id,ref,url), onCrop(id,url,ref), onEdit(id,ref,url), onSendToActiveTab(url), onShowToast(msg)
  output: {imageUrl, thumbnailUrl, imageUrlRef, imageAvailable, label}
  AI生成: 否 → 纯展示/上传节点（onChange 本地上传 → updateNodeData 回写 imageUrl）
  connected: 上游 target(Left) 接收 imageUrl；下游 source(Right) 输出 imageUrl → promptNode/gridSplitNode/cropNode/gridMergeNode

promptNode (Ya):
  props: {id, data{prompt,aspectRatio,imageSize,selectedModel,apiFormat,selectedContextResources,expanded,drawingModel,imageUrl,imageUrlRef,imageUrlThumbRef,thumbnailUrl,loading,errorMessage,onZoom,onCrop,onEdit,onSendToActiveTab,onShowToast,onAddImage,onGenerate,onStop,inputWidth,inputHeight,presetPrompts}, selected, width}
  callbacks: onGenerate(id,prompt,'1024x1024',model,apiFormat,batch), onStop(id), onZoom, onCrop, onEdit, onSendToActiveTab, onShowToast, onAddImage
  output: {prompt, aspectRatio, imageSize, selectedModel, apiFormat, selectedContextResources, expanded, inputWidth, inputHeight, imageUrl(+imageUrlRef/thumbnailUrl/loading/errorMessage 由父组件生成阶段回写)}
  AI生成: 是 → onGenerate → 网关 http://127.0.0.1:9004 → 回写 data.imageUrl / imageUrlRef / thumbnailUrl（loading/errorMessage 由父侧管理）
  connected: 上游 target(Left) 接收 imageUrl/text（来自 imageNode/imageBoxNode/videoNode/videoExtractNode/textNode/urlToImageNode/fileToUrlNode/panoramaNode）；下游 source(Right) 输出 imageUrl → gridSplitNode/cropNode/gridMergeNode

textNode (Qa):
  props: {id, data{prompt,text,autoSplit,expanded,selectedContextResources,selectedModel,textModel,loading,errorMessage,fontSize,inputWidth,inputHeight,presetPrompts,onShowToast,onAddImage,onGenerateText,onStop}, selected}
  callbacks: onGenerateText(id,prompt,autoSplit,model), onStop(id), onAddImage, onShowToast
  output: {prompt, text, autoSplit, expanded, selectedModel, selectedContextResources, inputWidth, inputHeight, fontSize(只读)}
  AI生成: 是 → onGenerateText → 网关生文 → 回写 data.text（loading/errorMessage 由父侧管理）
  connected: 上游 target(Left) 接收 imageUrl/text 素材；下游 source(Right) 输出 text → promptNode/gridSplitNode 等

cropNode (eo):
  props: {id, data{imageUrl, onCropComplete, onCancel}, selected}
  callbacks: onCropComplete(id, dataURL), onCancel(id)
  output: 不写 node data；通过 onCropComplete(id,croppedDataURL) 回传裁剪结果，由父组件落库/回写上游节点
  AI生成: 否 → 裁剪工具节点（通常以浮层方式打开，onCropComplete/onCancel 在调用点注入）
  connected: 上游 target(Left) 接收 imageUrl；下游 source(Right) 可选透传 imageUrl

gridSplitNode (po):
  props: {id, data{gridSize,rows,cols,splitMode('grid'|'manual'|'lasso'),hLines,vLines,lassoShapes,titlePattern,sendToImageBox,imageUrl,extractedImages,onSplit,onSplitOne,onSplitManual,onSplitOneManual}, selected}
  callbacks: onSplit(id,imageUrl,rows,cols,titlePattern,opts), onSplitOne(id,imageUrl,rows,cols,index,title,opts), onSplitManual(id,extractedImages,titlePattern,opts), onSplitOneManual(id,cellDataUrl,index,title,opts)
  output: {extractedImages, rows, cols, gridSize, splitMode, hLines, vLines, lassoShapes, titlePattern, sendToImageBox, imageUrl(从上游获取)}
  AI生成: 否 → 切图节点（canvas 本地裁剪，不入 AI 网关）
  connected: 上游 target(Left) 接收 imageUrl；下游 source 'batch'(Right) 整体输出 + source 'cell-N'(Right) 逐格输出 → gridMergeNode(cell-N)/imageNode

gridMergeNode (To):
  props: {id, data{gridSize,rows,cols,mergeMode('grid'|'longImage'|'overlay'),cellSize,aspectRatio,autoSize,titlePattern,longDirection,longGap,longTargetSize,longAutoSize,bgColor,overlayState,canvasWidth,canvasHeight,imageUrl,onSpawnImageNode,onShowToast}, selected}
  callbacks: onSpawnImageNode(id, mergedDataURL, 'merged-<mode>'), onShowToast(msg)
  output: {mergeMode, rows, cols, cellSize, aspectRatio, autoSize, titlePattern, longDirection, longGap, longTargetSize, longAutoSize, bgColor, overlayState, canvasWidth, canvasHeight, imageUrl(合成结果)}
  AI生成: 否 → 拼图节点（canvas 本地合成，合成后 onSpawnImageNode 派生新 imageNode）
  connected: 上游 target 'default'(Left) 顺序填充 + target 'cell-N'(Left) 定点填充（来自 gridSplitNode cell-N / imageNode）；下游 source 'merged-output'(Right) + source 'batch-output'(Right) 输出 imageUrl → imageNode/中转站
```
