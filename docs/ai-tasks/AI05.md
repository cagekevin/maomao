# AI05 — @ai-check 函数识别任务

> 识别以下 18 个 App.js 函数，提交 `混淆名 = 可读名 # 依据` 格式的映射。

## 方法

1. 对每个函数，grep 其在 App.js 中的定义和调用点
2. 读懂功能，给出语义化英文名（如 `handleSave`、`toastManager`）
3. 如果函数体简短（< 10 行），原文引用函数体作为"依据"
4. 如果有中文文案，提取关键文案辅助判断
5. 如果不确定，标注 `(?)` 表示存疑

## 输出格式

```
# 混淆名 = 可读名 # 依据
vr = processResourceItem  # 遍历 resources 数组对每项调 uploadFile
si = handlePromptApply(?) # 根据 responseBox 结果做节点回写，存疑
```
