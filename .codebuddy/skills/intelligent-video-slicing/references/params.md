# 参数调优规则参考

## 检测器选择

| 模式 | 类 | 适用场景 | threshold 范围 |
|------|-----|----------|----------------|
| `"adaptive"` | AdaptiveDetector(luma_only=True) | 静止背景 + 人物动作 (亮暗突变) | 1.5-12.0 |
| `"hsv"` | ContentDetector | 色彩丰富的场景，HSV 直方图比较 | 1.5-50.0 |

**默认使用 `"adaptive"`**，只有灰度版反复调参仍不理想时才换 `"hsv"` 作备用。

> 实测踩坑：本批达人混剪素材（静止背景 + 人物动作）用 hsv 反而更差——出现 21s/47.9s 超长漏切段。hsv 的 ContentDetector 阈值语义不同（默认 27，越低越灵敏），且对这种"人物微动但背景静止"的素材不匹配。在 hsv 参数没专门调好前，**不要**切到 hsv，老老实实用 adaptive。

## MIN_SCENE_LEN 按源时长分级

| 源视频时长 | MIN_SCENE_LEN | 原因 |
|-----------|---------------|------|
| ≤ 30s | 2s | 短原片允许切成更短的镜头，保节奏 |
| 超过 30s | 4s | 长原片用更长最小镜头，防碎片多、保连贯 |

## 切片质量判定

### 合格标准

- 切片数量 ≥ 2（不能整段一刀未切）
- 每个 clip 长度尽量在 4-8s 区间
- 平均时长在 3-8s 范围内（留一点弹性）
- **不含任何 >12s 的片段**（>12s 视为“一刀没切”硬伤，通常来自固定机位静止镜头）

### 欠切 (需降阈值)

- 切片数 = 1（整段未切）
- 或 平均 clip 时长 超过 8s
- 或 存在任意单片段时长 > 12s 的硬伤

动作：`threshold = max(1.5, threshold * 0.7)`

### 过切 (需升阈值)

- 平均 clip 时长 不足 2s

动作：`threshold = min(12.0, threshold * 1.5)`

### 收敛 / 放弃

- 调参 ≥ 3 次后仍不合格 → 放弃，标记 SKIPPED
- 阈值已触下限 (1.5) 且欠切 → 放弃
- 阈值已触上限 (12.0) 且过切 → 放弃
- **固定机位静止镜头**：含 >12s 硬伤段但降阈值（含 adaptive 1.5）和换 hsv 都无法切动时，直接 SKIPPED。这是素材本身无跳变点，不是参数问题，不要空耗重试次数。

## 初始默认值

```
DETECTOR_MODE       = "adaptive"
DEFAULT_THRESHOLD   = 3.0
DEFAULT_MIN_SCENE_LEN = 2
```
