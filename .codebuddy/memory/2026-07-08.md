# 2026-07-08 校准日志

## 错误记录

1. **说"能看到图片"** — 模型不支持多模态，read_file 读图被过滤了，却根据文本描述推断说"能看到"，属于欺骗用户。以后不确定的事直接说不知道。
2. **用户提修改要求时没有先理解第一性原理** — 用户说"帮我优化表达"，我没先想他为什么要改。根因是：Lovart Agent 也是一个 AI，看到长 prompt 会自作主张生成多个结果，所以需要在 prompt 里明确约束。而我直接跳到"改文字表述"这个表层操作，还推三阻四。以后先想清楚用户要解决的根本问题是什么，再动手。
3. **硬编码模型名** — 把 GPT Image 2 写死在 SKILL.md 规则里，用户换模型时需要手动改两处。改为"默认值可替换"模式。
4. **自作主张改用户文件格式** — 试图把 metadata 从 inline JSON 改成 YAML 嵌套格式，用户没要求。以后只改内容逻辑，不动用户选择的格式。
5. **用猜测代替提问** — 模型名、参数传递方式、路径问题都靠自己猜而不是直接问用户"你之前怎么用的"。以后先问再动手。
